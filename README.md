# Shape Up Website Lab

## Overview
This project is a responsive web page created for a web development lab assignment. The goal of the project was to practice building and modifying a website using HTML and CSS while implementing responsive design techniques.

## Features
- Semantic HTML5 page structure
- Responsive layout using CSS and media queries
- Flexbox layout for page organization
- Multi-level navigation menu
- Print stylesheet for formatting the page when printed

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries
- Git and GitHub
